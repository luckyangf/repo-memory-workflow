# 编辑器集成（可选）

使用 Skill / 项目规则 或快捷 Prompt 后，可直接说「拆一下」「继续」等短指令，无需每次粘贴长 prompt。

| 编辑器 | 方案 | 说明 |
|--------|------|------|
| **Cursor** | Rule / Skill | [cursor/README.md](cursor/README.md) |
| **TRAE** | 项目规则 | [trae/README.md](trae/README.md) |
| **VSCode + Codex** | Skill | [codex/README.md](codex/README.md) |
| **其他**（Copilot Chat、ChatGPT 等） | 快捷 Prompt | `init` 后项目内生成 `.ai/QUICK_PROMPTS.md`，复制粘贴使用 |
